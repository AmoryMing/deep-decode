# Power-ups（互动教程）

## 一句话定义

Power-ups 是 Claude Code 内置的 10 个互动课程——每个教一个大多数人没发现的功能，带动画演示。v2.1.90 引入。

## 大白话解释

就像游戏里的"新手教程"，但不是给新手的——每个都教一个你可能用了很久 Claude Code 都没发现的进阶功能。输入 `/powerup` 就能看到菜单。

## 10 个 Power-up

| # | 名称 | 教什么 |
|---|------|--------|
| 1 | Talk to your codebase | `@` 引用文件和行号 |
| 2 | Steer with modes | `Shift+Tab` 切换模式（plan/auto） |
| 3 | Undo anything | `/rewind` 和 `Esc-Esc` 回退 |
| 4 | Run in the background | 后台任务和 `/tasks` |
| 5 | Teach Claude your rules | CLAUDE.md 和 `/memory` |
| 6 | Extend with tools | [[MCP]] 和 `/mcp` |
| 7 | Automate your workflow | [[Skills]] 和 [[Hooks]] |
| 8 | Multiply yourself | [[Subagents]] 和 `/agents` |
| 9 | Code from anywhere | `/remote-control` 和 `/teleport` |
| 10 | Dial the model | `/model` 和 `/effort` 调模型 |

## 实操要点

1. **新人入门的最佳起点**：比读文档高效
2. **涵盖了 Claude Code 的核心能力图谱**：从基础引用到高级编排
3. **用法**：`claude` → `/powerup` → 选一个开始
