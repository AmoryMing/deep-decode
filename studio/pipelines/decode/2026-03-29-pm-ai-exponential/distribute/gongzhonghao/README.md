## 发布步骤

### 手动发布（推荐）

1. 登录公众号后台 https://mp.weixin.qq.com
2. 点击「内容管理」->「新建图文」
3. 设置标题（从上方备选标题选一个）
4. 正文：打开 `article.html`，全选复制（Ctrl+A, Ctrl+C），粘贴到编辑器
5. 图片替换：
   - 搜索 HTML 中的 `<!-- 插入图片:` 注释
   - 在对应位置点击「插入图片」，上传 `../material/pngs/` 目录下对应的 PNG 文件
   - 图片列表：
     - `01_四大操作变革.png`
     - `02_建造鸿沟.png`
     - `03_提效悖论.png`
   - original_page.png（原文截图）可选择是否插入
6. 封面图：上传 `cover.png`
7. 摘要：粘贴上方 Preview Text
8. 预览 -> 发送到手机检查排版 -> 发布

### 注意事项

- 公众号编辑器会自动过滤部分 CSS，粘贴后检查行高和字号是否正常
- 如果引用块（blockquote）样式丢失，手动选中文字 -> 点击编辑器的「引用」按钮
- 图片建议宽度设为「100%」，不要用固定像素值
- 发布前用手机预览，确认中文换行和图片显示正常

### API 自动发布（高级）

微信公众号提供了素材管理和群发 API，但有以下限制：

- 需要认证服务号（订阅号无群发接口权限）
- API 文档：https://developers.weixin.qq.com/doc/offiaccount/Message_Management/Batch_Sends_and_Originality_Checks.html
- 流程：上传图片素材 -> 上传图文素材 -> 群发
- access_token 有效期 2 小时，需定期刷新
- 每月群发次数有限（认证服务号 4 次/月，订阅号 1 次/天）

如果后续需要批量发布，可以用 Python `requests` 库封装一个脚本，调用上述 API 完成自动化。
