# PowerPoint 核心功能

Claude for PowerPoint 的所有核心能力详解。每个功能包含：做什么、怎么用、注意什么。

---

## 1. 从模板构建

### 功能描述

Claude 会先读取你演示文稿中的幻灯片母版（Slide Master，控制全局样式的底层模板）、布局（Layout）、字体和配色方案，然后生成的每一张幻灯片都自动符合这些模板要求。

你不需要告诉 Claude "用什么字体""标题放哪"，它会从母版里自己读取。

### 示例 Prompt

| 你想做的事 | 对 Claude 说 |
|------------|-------------|
| 市场分析 | "创建市场规模分析 -- 3 张幻灯片涵盖 TAM、SAM、SOM 配可视化" |
| 执行摘要 | "使用单栏内容布局添加执行摘要幻灯片" |
| 竞品对比 | "建一张竞品对比幻灯片，比较 4 家竞争对手" |

### 使用提示

- 先打开包含母版和配色方案的模板文件，再让 Claude 生成内容
- Claude 会自动选择最合适的布局，但你也可以指定用哪个布局
- 如果对生成结果不满意，可以直接说"换个布局"或"调整配色"

---

## 2. 编辑现有幻灯片

### 功能描述

选中某张幻灯片，用自然语言描述你想做的修改。Claude 会在编辑时保留原有的格式、字体、配色和上下文，不会因为改一段文字就把整张幻灯片的样式搞乱。

支持单张编辑，也支持跨多张幻灯片的批量调整。

### 示例 Prompt

| 你想做的事 | 对 Claude 说 |
|------------|-------------|
| 精简文字 | "简化这张幻灯片的文字" |
| 添加图表 | "添加展示季度趋势的图表" |
| 重构叙事 | "重构第 4-7 张幻灯片的故事线" |
| 合并幻灯片 | "把第 5 和第 6 张合并为一张总结" |

### 使用提示

- 编辑前 Claude 会警告是否覆盖已有内容
- 所有编辑都可以用 PowerPoint 标准撤销（Ctrl+Z / Cmd+Z）
- 说"简化"Claude 会精简文字；说"扩展"会补充细节 -- 用词要明确

---

## 3. 生成完整演示文稿

### 功能描述

从一段自然语言描述出发，Claude 可以一次性生成一套完整的演示文稿，包括：
- 幻灯片结构和顺序
- 每张幻灯片的标题、正文、要点
- 图表和可视化（原生 PowerPoint 对象，不是图片）
- 过渡页和章节分隔

### 示例 Prompt

| 你想做的事 | 对 Claude 说 |
|------------|-------------|
| 市场进入 | "创建 10 张幻灯片走查我们的市场进入假设" |
| 项目更新 | "构建内部项目更新演示，含时间线和后续步骤" |
| 融资路演 | "做一套 15 页的融资路演 PPT，涵盖问题、方案、市场、团队" |

### 使用提示

- 生成完整演示后，建议先整体看一遍结构，再逐张调整细节
- 可以分步指导："先做前 5 张关于市场的，我看一下再做后面的"
- 生成的幻灯片数量和你要求的可能略有出入，Claude 会根据内容量自动调整

---

## 4. 原生图表和图形

### 功能描述

Claude 把要点转换为可编辑的原生 PowerPoint 对象 -- 图表（Chart）、流程图（Process Flow）、图形（Diagram）。关键区别：这些是原生 PowerPoint 对象，不是截图或静态图片。

这意味着你生成之后还可以：
- 双击图表修改数据
- 调整图表类型和样式
- 拖拽流程图的节点

### 示例 Prompt

| 你想做的事 | 对 Claude 说 |
|------------|-------------|
| 数据可视化 | "把这些要点转换为流程图" |
| 柱状图 | "用这个数据表创建柱状图" |
| 组合图表 | "做一个收入柱状图加利润率折线的组合图表" |

### 使用提示

- 生成的图表可以在 PowerPoint 中直接编辑，和手动创建的图表一样
- 如果数据量大，建议先让 Claude 在 Excel 中处理好数据，再跨应用生成图表（见 [[跨应用协作]]）

## 5. 模板感知

### 功能描述

Claude 会自动读取并尊重当前演示文稿中的以下设置：

| 模板元素 | Claude 怎么用 |
|----------|--------------|
| 幻灯片母版（Slide Master） | 遵循全局样式规范 |
| 布局（Layout） | 选择合适的布局生成新幻灯片 |
| 字体（Font） | 使用模板指定的字体族 |
| 配色方案（Color Scheme） | 图表、文字颜色与模板一致 |

你不需要每次都提醒"用公司模板的颜色"，Claude 自动执行。

### 使用提示

- 模板感知的前提是你的模板本身设置正确（母版和布局都配好了）
- 如果生成的幻灯片样式不对，先检查模板的母版设置是否完整

---

## 6. 连接器支持和 Skills 集成

### 功能描述

**连接器（Connector）：** 通过 MCP 连接器（Model Context Protocol，让 AI 调用外部数据的标准协议），Claude 可以从外部工具拉取上下文到 PowerPoint。比如从金融数据平台拉数据直接做图表。在 Claude 设置中配置的连接器会自动在 PowerPoint 中生效。

**Skills（技能）：** 在 Claude 设置中启用的 Skills 也会自动在 PowerPoint 中可用。在侧边栏输入 "/" 可以查看当前可用的 Skills 列表。

Skills 可以帮你：
- 自动应用建模规范
- 匹配行业特定的模板要求
- 执行标准化的分析流程

### 使用提示

- 连接器和 Skills 的配置在 Claude 的全局设置中管理，不在 PowerPoint 内部
- 自定义连接器可能引入安全风险，启用前请评估来源可信度

---

## 7. 持久指令

### 功能描述

在 PowerPoint 加载项的 Instructions（指令）字段中，你可以设置每次 PPT 对话都自动生效的偏好。比如：

- "所有幻灯片标题用中文"
- "图表配色优先使用公司品牌色"
- "每张幻灯片正文不超过 5 个要点"

这些指令会在每次新对话时自动加载，不需要重复说。

### 使用提示

- PowerPoint 的持久指令和 Excel 的持久指令是分开管理的，互不影响
- 指令写得越具体越好，含糊的指令（如"做得好看点"）效果不稳定
- 可以随时修改指令，修改后立即对新对话生效
