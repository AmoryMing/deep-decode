# Release Notes - Claude for Excel & PowerPoint
> Source: https://support.claude.com/en/articles/12138966-release-notes (filtered for Office)
> Crawled: 2026-04-10

## March 11, 2026
**Work across Excel and PowerPoint**
- Claude for Excel and PowerPoint add-ins now share full conversation context
- Actions in one application are informed by activity in the other
- Support for skills added to both add-ins
- Amazon Bedrock, Google Cloud Vertex AI, and Microsoft Foundry users can connect via LLM gateway

## February 5, 2026
**Claude for PowerPoint Introduction**
- Claude is now available as an add-in for PowerPoint
- Enables presentation creation and editing directly within Office

**Claude for Excel Improvements**
- Updated to use Opus 4.6 model
- Supports native Excel operations including pivot table editing
- Enables conditional formatting functionality

## November 24, 2025
**Claude for Excel Beta Release**
- Available to Max, Team, and Enterprise users
- Additional capabilities: pivot tables, charts, file uploads
- Keyboard shortcut (ctrl+option+c) to open full Claude app from Excel
- Performance and context management enhancements
