## 发布顺序建议

1. **公众号**（图文长文，作为内容锚点，其他平台可引流回公众号）
2. **B站**（长视频，深度内容用户会主动搜索）
3. **视频号**（社交分发，发布后立刻分享到微信群触发初始传播）
4. **小红书**（搜索+推荐双引擎，标签策略拉长尾流量）
5. **抖音**（算法推荐，前3秒决定生死，放在最后可以根据前几个平台反馈微调钩子）
